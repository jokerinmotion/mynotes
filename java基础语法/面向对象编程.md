### 类的结构：属性和方法

#### 类

类是一组属性（成员变量）和行为（成员方法）的集合。是抽象的。

类的创建

```java
public class 类的名称{
    （一个类里面的）多种成员变量;//属性声明
    (类里面含有的)多种成员方法，定义的时候没有static;//方法声明
}
```

#### 属性（成员变量）

属性，也是成员变量，= field、域等等名称

属性没有初始化之前的默认值和一维数组的默认初始化值一样。
