- java集合
  - Collection接口：单列数据，定义了存取一组对象的方法的集合
    - List接口：元素有序、可重复的集合。也称为**“动态数组”**。
    - Set接口：元素无序、不可重复的集合。
    - 还有其他接口

  <img src="images/image-20210729133956340.png" alt="image-20210729133956340" style="zoom:50%;" />

  - Map接口：双列数据，保存具有映射关系“key-value对”的集合
