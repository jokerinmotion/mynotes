## 快捷键

- controll shift T：查找JDK的源代码
- ctrl O：查看代码的结构
- alt shift s : 生成代码块
- alt shft / ：查看方法和构造器行参
- ctrl alt  T ：try-catch等代码块包围u
- ctrl+shift+/ 这个是多行代码注释在一个块里，只在开头和结尾有注释符号
- alt shift + M ：提取方法
- alt shift + V ：提取变量
